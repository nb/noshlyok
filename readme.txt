Не позволява изпращането на коментари без поне един кирилишки символ.

== Description ==

Без шльокавица ви пази по от нежеланиете коментари, като се възползва от факта,
че повечето спам не е на кирилица.

Ако все пак решите да напишете публикация, на която желаете да бъдат позволени
коментарите с не-кирилишки символи, може да го разрешите от страничната лента
на всяка публикация. 

== Installation ==

1. Качете файла `noshlyok.php` в `/wp-content/plugins/` на вашия сървър
1. Включете разширението от менюто 'Разширения' (Plugins)
1. Изключете другите анти-спам разширения :-)
